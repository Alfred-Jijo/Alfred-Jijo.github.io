◆ gpalloc()

function Allocator gpalloc ( void )
inline

Constructs a gpalloc Allocator with default options.

Equivalent to gpalloc_ex(GPALLOC_DEFAULTS). This is the function you'll call 95% of the time.

Example:

Allocator a = gpalloc();
Node *n = allocator_alloc(&a, sizeof(Node));
allocator_free(&a, n, sizeof(Node));
gpalloc_destroy(&a);

◆ gpalloc_destroy()

function void gpalloc_destroy ( Allocator * a)
inline

Frees the internal context object allocated by gpalloc / gpalloc_ex.

Call this when you're done with the Allocator itself (not its allocations — those are your responsibility). Safe to call with a zeroed Allocator.

Parameters
aPointer to the Allocator to destroy.

◆ gpalloc_ex()

function Allocator gpalloc_ex ( GpAllocOptions opts)
inline

Constructs a gpalloc Allocator with explicit options.

The internal context object is itself heap-allocated. Free it with gpalloc_destroy() when the Allocator is no longer needed.

Parameters
optsConfiguration options (use GPALLOC_DEFAULTS as a base).
Returns
A configured Allocator, or a zeroed struct on OOM.
