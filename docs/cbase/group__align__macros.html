◆ IS_POW2

#define IS_POW2 ( x)
Value:
((x) != 0 && (((x) & ((x) - 1)) == 0))

Checks if a given integer is a strictly positive power of 2.

Unlike IS_POW2_OR_ZERO(), this returns false for 0. Use this when validating alignment values, since an alignment of 0 is never valid.

Parameters
xThe value to check.
Returns
1 (true) if x > 0 and is a power of 2, 0 (false) otherwise.
