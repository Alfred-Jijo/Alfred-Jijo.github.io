Detailed Description

General-purpose allocator (gpalloc) — malloc/free backed Allocator.

A ready-made Allocator that wraps the C standard heap. Use it as a drop-in default when no Arena or pool is available, or as a fallback allocator in tests.

Zero-initialisation behaviour is controlled at compile time: -DGPALLOC_NO_ZERO → use malloc (faster, no zero guarantee) (default) → use calloc (zero-initialised, matches arena behaviour)

Usage — default (zero-init):
Allocator a = gpalloc();
MyStruct *s = allocator_alloc(&a, sizeof(MyStruct));
allocator_free(&a, s, sizeof(MyStruct));
