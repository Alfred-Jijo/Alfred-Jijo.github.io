Detailed Description

Accumulate strings efficiently and join them into a single allocation.

Typedef Documentation

◆ StringBuilder

typedef struct StringBuilder StringBuilder

An arena-backed string builder.

The builder does not own any memory directly. All allocations are made through the arena(s) passed to each operation.

  • Example (The Scratch Arena Pattern):
    MemArena scratch = mem_arena_create(MB(1));
    MemArena result_arena = mem_arena_create(MB(4));
    StringBuilder sb = {0};
    sb_append(&sb, &scratch, STR8("Hello, "));
    sb_append_fmt(&sb, &scratch, "%s!", "World");
    String8 result = sb_join(&sb, &result_arena);
    // result.str is now safely inside result_arena.
    // The linked-list node overhead is completely wiped out without fragmentation!
    mem_arena_clear(&scratch);
