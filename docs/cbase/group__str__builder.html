◆ sb_join()

String8 sb_join ( StringBuilder * sb,
Arena * out_arena )
inlinestatic

Materializes all appended segments into a single contiguous String8.

Allocates one buffer of total_size bytes from out_arena, copies every segment into it in order, and returns the result. The returned string is NOT null-terminated unless the last appended segment ended with a NUL byte.

The builder's node list is left intact after this call. Call sb_reset() if you want to reuse the builder.

Parameters
sbThe builder whose segments to join.
out_arenaArena that will own the final contiguous string.
Returns
A String8 pointing into out_arena, or a zero String8 if the builder is empty or OOM.

void sb_reset ( StringBuilder * sb)
inlinestatic

Resets the builder to empty without touching the arena.

Clears the node list pointers and counters. The arena memory used by previous nodes is not reclaimed — use an ArenaTemp around the whole build-and-join sequence if you want to reclaim it.
