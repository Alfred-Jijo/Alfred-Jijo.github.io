Detailed Description

High-performance Virtual Memory Arena Allocator.

Allocates memory by linearly advancing an offset within a large reserved virtual memory block. Automatically commits physical RAM pages from the OS as needed. Features zero-overhead allocations and instant mass-deallocation.
