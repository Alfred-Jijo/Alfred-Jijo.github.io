# Portfolio Website

This is a portfolio website that showcases my projects, skills, and background. It includes links to my GitHub profile and provides a way for visitors to contact me.

## Project Structure

- `src/css/styles.css`: Contains the styles for the portfolio website.
- `src/js/main.js`: Contains the JavaScript code for interactivity.
- `src/pages/about.html`: Provides information about me.
- `src/pages/contact.html`: Includes my contact information.
- `src/pages/projects.html`: Showcases my projects with links to my GitHub profile.
- `index.html`: The main entry point for the website.
- `package.json`: Configuration file for npm.

## Setup and Usage

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd portfolio-website
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Open `index.html` in your browser to view the portfolio website.
